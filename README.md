# Online Voting System

## Project Overview
This is an Online Voting System designed to allow users to securely vote through a web platform. The system ensures transparency, security, and ease of use. It is built using a .NET backend and a modern frontend framework, with proper integration between both layers.

## Tech Stack
- Backend: ASP.NET Core Web API
- Frontend: React / JavaScript
- Database: SQL Server (or other relational DB) [Only during development phase]
- Authentication: JWT (JSON Web Token)
